******API******
*************************************************************************************
1)open LocationTrackerAPI in Visual studio then Run.
2)Provide inputs to API which appear with swagger or call api using other apps.
3)sample API - https://localhost:44322/api/LocationTracker?truckId=TR00056&temperature_celcius=34%20C&latidude=20.2046%C2%B0%20N&longitude=70.4977%C2%B0%20E
  Note: change your port accordingly to your localhost.
4)Will be getting response message as "Details saved Successfully" once data saved.
5)Please Refer APIOutputScreenShot! 
*************************************************************************************


******WEB UI******
******************************************************************************************
1)open LocationTrackerWEB in Visual studio then Run.
2)You would be able to see the truck details data which has been saved in API calling Process.
3)Keep calling API with different location data, simultaneously the data would be reflecting in UI.
